calcBonusFactor Source #

Arguments

:: String

Answer for which the multiplier is calculated

-> [String]

List of all Answers (including the one being evaluated)

-> Int

Score multiplier (1 | 2 | 4)

Calculates a score multiplier for a given Answer 4: if only one non empty answer was given. (No one else answered in that category) 2: The answer is unique. (There where other answers, but not equal answers) 1: The answer was given multiple times. Note: Empty answers will get score multipliers too, but since they will receive 0 points the multiplier does not matter
